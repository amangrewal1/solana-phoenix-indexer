# Contributing

Thanks for your interest!

## Setup

```bash
pip install -r requirements.txt
```

## Running tests

```bash
make test
```

## Style

Keep functions small and focused. Add a brief docstring for any function
that isn't self-explanatory from its name. Ruff / black for formatting.

## Scope

This is a focused research/portfolio project. Issues and PRs for clear bugs
or measurement mistakes are welcome; feature requests outside the stated
scope may not be acted on.
